# <p align="center">Windsurf IDE</p>

**Windsurf** ist unsere bevorzugte IDE für die Entwicklung im NADOO-Universum. Sie basiert auf VS Code, integriert aber tiefgreifende KI-Funktionen ("Cascade"), die unsere Arbeitsweise revolutionieren.

## Warum Windsurf?

- **Cascade Flow:** Ein integrierter KI-Agent, der den Kontext des gesamten Projekts kennt.
- **Workflows:** Automatisierte Abläufe für Standard-Tasks (Feature-Erstellung, Bugfixing).
- **Memories:** Persistentes Gedächtnis für Projekt-Regeln und Konventionen.

## Einrichtung

Siehe [Installation & Einrichtung](../02-vscode/01-installation/README.md) (wird dort behandelt).

## Wichtige Konzepte

- **Memories:** `.windsurf/memories/` speichert Regeln.
- **Workflows:** `.windsurf/workflows/` definiert Prozesse.
